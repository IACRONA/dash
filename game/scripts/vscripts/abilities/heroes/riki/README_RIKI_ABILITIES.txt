================================================================================
СИСТЕМА СКРЫТЫХ СПОСОБНОСТЕЙ RIKI
================================================================================

ОПИСАНИЕ:
При входе Riki в невидимость у него появляется дополнительный бар из 4 способностей.
Уровни способностей синхронизируются с основными (качаешь первую - качается первая в скрытом баре).

РЕАЛИЗОВАННЫЕ КОМПОНЕНТЫ:

1. riki_invisibility_manager.lua
   - Пассивная скрытая способность, управляющая сменой способностей
   - Отслеживает состояние невидимости героя
   - Меняет способности при входе/выходе из невидимости
   - Синхронизирует уровни способностей

2. riki_stun_strike.lua
   - Первая способность скрытого бара
   - Стан на 1/2/3/3 секунды
   - Урон: 75/125/175/225
   - Мана: 100/150/150/150
   - Перезарядка: 15/10/10/8 секунд
   - Иконка: bounty_hunter_shuriken_toss

ИЗМЕНЁННЫЕ ФАЙЛЫ:

1. scripts/npc/npc_heroes_custom.txt
   - Добавлен Riki с управляющей способностью в слоте Ability7
   - Подготовлены слоты Ability10-13 для скрытых способностей

2. scripts/npc/npc_abilities_custom.txt
   - Добавлено определение riki_invisibility_manager
   - Добавлено определение riki_stun_strike

КАК РАБОТАЕТ:

1. При выборе Riki автоматически добавляется скрытая способность riki_invisibility_manager
2. Эта способность создаёт модификатор, который проверяет состояние невидимости каждые 0.1 секунды
3. При входе в невидимость первая способность (Smoke Screen) меняется на Stun Strike
4. При выходе из невидимости способности возвращаются к обычным
5. Уровни способностей автоматически синхронизируются

ДЛЯ ДОБАВЛЕНИЯ НОВЫХ СПОСОБНОСТЕЙ:

1. Создайте новый файл способности в папке abilities/heroes/riki/
2. Добавьте определение в npc_abilities_custom.txt
3. Обновите массив hidden_abilities в riki_invisibility_manager.lua
4. Добавьте логику синхронизации уровней в RegisterLevelUpEvent
